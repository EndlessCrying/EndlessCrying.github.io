# “摩羯”简介
# 小恐慌
大概在3日左右，正在学校“坐牢”的师生们注意到了今年第十一号台风摩羯的生成。
不过作为一个~~成熟的~~海南人，对于台风早已习以为常了。
每年总有几个台风登录海南，呼哧呼哧地带来一阵风一场雨，但也就仅此而已。
所以谁也没把刚出生的小摩羯放在心上，大家都认为是个普通的小台风。
不过事态还是朝着最坏的情况发展，摩羯三天连升五级，直奔着超强台风级别而来。
